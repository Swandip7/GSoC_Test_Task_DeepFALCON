# GSoC 2026 DeepFalcon Test Task Submission

This repository contains my complete Jupyter notebook solution for the GSoC 2026 DeepFalcon tasks. It covers the full pipeline from image-based representations to graph-based learning for quark/gluon jet events.The notebook includes data preprocessing, Beta-VAE training for multi-channel jet image reconstruction (ECAL, HCAL, Tracks), and visualization of original vs reconstructed events. It also implements graph construction from point cloud representations and trains a GNN (GAT-based) model for jet classification.In addition, a graph autoencoder is trained to reconstruct node-level features, with side-by-side visual comparisons and evaluation metrics. The work also includes latent space analysis using PCA, training curves, and a discussion of how architectural choices (such as k-neighbors, hidden dimensions, and edge features) impact performance.
